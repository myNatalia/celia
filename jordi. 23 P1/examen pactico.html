3. Párrafo Justificado

Este es un ejemplo de texto justificado. El texto justificado se alinea tanto en el margen izquierdo como en el derecho, creando líneas rectas en ambos lados. Esto se utiliza comúnmente en periódicos, revistas y libros para crear un aspecto más organizado y formal. Para lograr este efecto, utilizamos la propiedad CSS text-align con el valor justify.
